= Configure iSCSI Boot
include::entities.adoc[]

In this step you will configure the iSCSI initiator service on both nodes to start on boot.

[#procedure-configure-iscsi-boot]
.Procedure: Configure iSCSI Boot

. On both of the nodes that will be running {susemgr} run:
+

----
sudo yast iscsi-client
----

. On the service tab select the service to be started on boot:
+

image::step1-iscsi.png[Start iSCSI on boot]

. Make note of the initiator names. For example my node initiator names were:
+

----
node1: iqn.1996-04.de.suse:01:68ae1d6bb09f
node2: iqn.1996-04.de.suse:01:2afa2e93d6a
----

. Proceed to the next section to setup the storage server files.
