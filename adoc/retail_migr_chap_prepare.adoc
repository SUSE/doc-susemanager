== Migration Scenarios

This document contains several different migration scenarios.
Ensure you pick the most appropriate migration scenario for your environment.

If you are migrating from any version of {slepos} to {smr}, and you do not have a current {susemgr} installation, see: xref:retail_migr_slepostosuma.adoc#retail_migr.chap.slepostosuma[SLEPOS to new SUSE Manager]

If you are migrating from any version of {slepos} to {smr}, and you already have a current {susemgr} installation that you wish to use, see: xref:retail_migr_slepostosumaind.adoc#retail_migr.chap.slepostosumaind[SLEPOS to Independent SUSE Manager]
